📌 이번 강의 핵심 메시지

사업계획서의 '문제(Problem)'는 사회 문제가 아니라 내가 해결할 수 있는 고객의 문제여야 한다. 외부 동기 + 내부 동기를 조합하고, AI 도구로 빠르게 근거를 만들어라.

📢
강의 시작 안내

이번 주 진행 상황

  • 특허 주제를 피치스파트너스 게시판에 제출한 분들 — 강사 피드백 달아드리는 중
  • 출원 후 1~2주 내 결과 확인 → 출원 사실 증명서 발급 → 사업계획서 삽입
  • 출원번호만 나온 경우: 출원번호 + 제목 + 주요 내용 요약본 삽입 가능
  • 과제명 방향성 나온 분들은 바로 사업계획서 작성 시작!
"불완전해도 일단 시작하라. 완벽한 아이디어를 기다리다 기회를 놓친다."
🎙️
왜 라이브 강의인가

지원사업 공고는 실시간으로 변한다. 어제 마감된 사업, 오늘 갑자기 나온 추경 사업이 있다. 녹화 강의로는 이 정보를 따라갈 수 없다.


→ 라이브로 최신 공고 정보를 같이 확인하고 바로 지원 전략을 세우는 것이 핵심

❗
P(Problem) — 문제 정의의 핵심
❌ 탈락하는 방식

사회 문제를 쓰는 것

"저출산 문제가 심각합니다."
"노인 인구가 증가하고 있습니다."
"환경 오염이 심각합니다."

→ 평가위원들이 가장 많이 보는 뻔한 패턴. 내가 어떻게 해결할 수 있는지 불분명.

✅ 합격하는 방식

내가 해결 가능한 고객의 문제

외부 동기: 특정 고객군이 겪는 구체적 불편함
내부 동기: 내가 이 문제를 해결해 온 노력·자랑

→ 두 가지를 조합하면 설득력이 완성된다

💡 외부 동기 + 내부 동기 공식
1
외부 동기 — 고객의 문제

경쟁사 제품을 찾고 → 한계/불만을 발견하고 → 그게 특정 고객군의 진짜 문제임을 증명

2
내부 동기 — 내가 해온 노력

이 문제를 해결하기 위해 내가 이미 해온 것들: 경력, 연구, 테스트, 설문 등

3
문제 ↔ 솔루션 1:1 매칭

각 문제에 대한 나의 해결책을 하나씩 대응시켜야 한다. 뭉뚱그린 해결책 NG

🔬
실습 예시 — UV 선 패치
📌 선 패치 제품 Problem → Solution 적용 흐름
①
경쟁사 찾기
기존 선크림, 선스프레이, 유사 선패치 제품 조사 (쿠팡, 아마존, 올리브영 등)
②
기존 제품의 한계 발견
미관 불량 (붙이면 티남) / 제거 시 통증 (피부 자극) / 트러블 유발
→ 이게 고객의 진짜 문제
③
1:1 솔루션 매칭
미관 문제 → 투명/패션 디자인 패치
통증 문제 → 저자극 접착 소재
트러블 문제 → 피부과 테스트 완료 성분
④
타겟 고객 명확화
야외 러닝·마라톤 인구 증가 → 피부에 민감한 20~40대 운동 마니아
"평가위원들에게 보여줄 구체적 증거가 필요하다. 설문, 테스트, 이미지. '될 것 같습니다'는 안 된다."
🤖
AI 도구 활용법 — "사람 냄새 나는 AI 작업"
🟢 ChatGPT / Claude
경쟁사 분석, 솔루션 도출, 내용 생성
내 아이템과 경쟁사 정보를 주면 한계·차별점·솔루션 자동 정리. 사업계획서 문장도 생성.
🔵 Lovable (러버블)
홈페이지 자동 생성
텍스트 입력만으로 서비스 홈페이지 자동 생성. URL이 생기면 사업계획서에 근거로 삽입 가능.
🟡 Notebook LM
슬라이드 · 인포그래픽 생성
문서를 업로드하면 슬라이드·인포그래픽 자동 생성. 도식화된 이미지를 사업계획서에 삽입.
🔴 핵심 원칙
AI가 만든 내용도 내 검증이 필요
AI 생성 텍스트만으로 채운 계획서는 평가위원이 바로 알아본다. 구체적 근거·이미지·수치를 추가해야 살아남는다.
🖼️ 도식·이미지 활용 팁
  • ChatGPT로 만든 내용 → Notebook LM에서 인포그래픽 변환 → 사업계획서 삽입
  • 경쟁사 제품 이미지 캡처해서 비교표에 넣기
  • Lovable로 만든 홈페이지 스크린샷 → 기술·서비스 구조도 대체
  • 평가위원은 글보다 이미지와 도식을 먼저 본다
📄
특허 출원 실습 — 특허로 사이트
STEP 1
특허로(patent.go.kr) 접속 → 임시 명세 출원 선택

정식 명세서 없이도 가능한 임시 출원 방식. 빠른 선점이 목적.

STEP 2
발명 명칭(한글) + 영문 명칭 입력

명칭은 이미 출원한 분들의 피드백으로 개선. 강사가 게시판에 피드백 중.

STEP 3
설명 파일 첨부

아이디어 설명 문서를 첨부 (PPT, PDF, 워드 등 가능). 완벽하지 않아도 됨.

STEP 4
수수료 납부 — 13,800원 (70% 할인 적용)

예비창업자 할인 적용 시 부담 없는 금액. 출원 직후 출원번호 발급.

STEP 5
출원 사실 증명서 발급 → 사업계획서 삽입

1~2주 후 지식재산센터 평가 후 청구범위 안내 옴. 증명서 출력해서 삽입.

💡 특허 주제 몇 건이 이상적?

최소 3건 권장. 출원번호가 3개면 사업계획서에 근거가 훨씬 탄탄해진다. 출원 사실 증명서가 나오기 전까지는 출원번호 + 제목 + 요약만 넣어도 OK.

👁️
평가위원 관점

하루 100개 심사

평가위원은 하루에 수십~100개를 검토한다. 첫 페이지부터 차별화되지 않으면 기억에 남지 않는다.

구체적 증거가 핵심

설문 데이터, 테스트 결과, 경쟁사 이미지, 업무협약서 — 말이 아닌 증거로 설득해야 한다.
